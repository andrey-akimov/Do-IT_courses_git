# Практическое задание

В ветке `master` содержится модуль с классом от которого наследуется класс в другом модуле

В ветке `fix` содержится код который исправляет ошибку в коде ветки master

Задача установить последнюю версию [node.js](https://nodejs.org/en/) с помощью которой нужно будет проверить работоспособность кода используя дополнительный аргумент `--experimental-modules` и с помощью команды `merge` выполнить слияние двух веток, так, чтобы код работал корректно. Затем создать конфликт на основе этого примера. И отправить pull reuest с любым кодом который реализует дополнительный функционал к этим классам.
